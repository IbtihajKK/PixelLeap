About PixelLeap

PixelLeap is an arcade-style 2D platform game developed as a semester project. The objective is to navigate through challenging levels by jumping, avoiding hazards, and reaching the finish point. The game focuses on timing, skill, and quick reflexes.


Features:

• Smooth jump & movement controls

• Level progression (Level 1, Level 2, Practice Mode)

• Portals, moving obstacles, and death detection

• Pause menu, level complete UI, and sound effects

• Responsive UI for Android screens

• Custom artwork + polished buttons
